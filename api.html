Basic API allow users to retrieve any info about the entire Epocum network


get_Proposals

get_Connections_list

http://localhost:88/get_Proposals

Retrieve all the proposals generated from every advertiser 

http://localhost:88/get_Acceptances

Retrieve all the acceptances of the users from each relative proposals 

http://localhost/get_Target/:proposal_hash

Retrieve update target of the visits of the current proposal's hash passed 

http://localhost/get_Connections_num/:acceptance_hash

Retrieve the current post visit of the current acceptance's (or easily the post you had share) 

http://localhost/get_Connections_list/:acceptance_hash

Retrieve all the visit's info of the current acceptance's (or easily the post you had share)
